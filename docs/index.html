🚀 How It Works

  1. Upload your photo via the /api/upload endpoint
  2. Generate fashion items using AI via /api/items/:category
  3. Remix your photo with selected items via /api/remix
  4. Download your transformed image from /api/outputs

🛠️ Technology Stack

• Rust - Systems programming language
• Axum 0.8 - Modern web framework
• Reqwest 0.12 - HTTP client
• Google Gemini AI - Image generation
• Blake3 - Cryptographic hashing
• Tokio - Async runtime
• Serde - Serialization framework
• Rust-Embed - Static file serving

⚙️ Configuration

Configure the application using these environment variables:

  • GEMINI_API_KEY - Your Google Gemini API key (required)
  • MIRAGE_FIT_PORT - Server port (default: 3000)
  • MIRAGE_FIT_HOST - Server host (default: 127.0.0.1)
  • RUST_LOG - Log level (default: info)
